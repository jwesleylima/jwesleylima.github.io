Ajuda na instalação

Esta sessão descreve o passo à passo para instalar o My Adventurous Dino corretamente. Se possível, verifique se a sua questão está registrada na Central de Ajuda.

Respeite todos os passos, assim nenhum erro é esperado.

Requisitos mínimos de hardware

Veja abaixo a lista de requisitos mínimos para rodar bem o jogo. Se sua máquina não se encaixa, não se preocupe. Estes dados são apenas especulações, o jogo pode se comportar de formas diferentes dependendo da sua máquina.

  1. Windows 7/8/8.1/10
  2. Processador dual core 1GHz
  3. 2GB RAM mínimo
  4. Monitor resolução 1280x766
  5. Memória HDD, SSD de 25GB mínimo
  6. 500MB VRAM mínimo
  7. OpenGL 1.1 mínimo

NOTA: Não há suporte para versões do Windows XP e Vista, pois não pudemos testar. Talvez rode, mas não temos como oferecer suporte avançado.

Primeiros passos

Baixe o arquivo ZIP do jogo clicando no link no cartão logo acima. Após isso, extraia os arquivos do jogo em uma pasta livre, isto é, que não requer permissão de administrador para acesso e modificação, já que o jogo precisa criar alguns arquivos de dados ao ser executado pela primeira vez.

IMPORTANTE: Atente-se que, mesmo em uma pasta livre, seu antivírus pode impedir que o jogo crie os dados necessários. Isso pode ser mudado nas configurações do programa de antivírus.

Pós extração

Agora que você extraiu os arquivos do jogo para uma pasta livre, é recomendável que você reinicie a sua máquina, apesar de que em nossos testes isto se mostrou desnecessário. Enfim, dê um duplo clique sobre o "My Adventurous Dino" e espere alguns segundos; após isso, verifique se a reação da sua máquina está descrita abaixo.

  • Nada acontece

    • Se esse for o seu caso, é possível que seja necessário reiniciar a sua máquina. Pode ser ainda, um bloqueio causado pelo antivírus como enfatizamos mais acima.

  • Apareceu uma mensagem de erro

    • Se esse for o seu caso, é possível que o erro esteja relacionado ao código-fonte, mas pode estar relacionado também com falta de arquivos ou arquivos com nomes alterados manualmente. Veja a relação de erro e motivo abaixo:

    No such file or directory
    Faltando arquivos necessários
    PermissionError
    Jogo instalado num local privado
    NameError
    Erro interno no código
    SDL2 Audio Device
    Seu dispositivo de áudio não foi reconhecido (reinicie o computador)

  • Apareceu uma janela preta

    • Se no seu caso uma janela preta surgiu e depois de alguns segundos desapareu, então temos um problema. Isso pode ocorrer quando você não reiniciou sua máquina. Pode ainda ocorrer decorrente de um bloqueio do seu programa de antivírus, que bloqueia o jogo de criar os arquivos de dados mandatórios.

    Isso pode ser mudado permitindo ao My Adventurous Dino modificar o diretório no qual foi instalado. Faça isso nas configurações do seu programa de antivírus.

Finalizando

Espero que você tenha conseguido jogar enfim. Pedimos desculpas por qualquer inconveniente.
