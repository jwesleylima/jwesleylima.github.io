Central de ajuda

PermissionError ao executar o jogo

Se quando você executa o jogo, aparece uma pequena janela apresentando um erro de permissão, este é seu problema. Para resolver isso, siga os passos abaixo:

  1. Desinstale o jogo

    2. Para isso, Execute o Uninstall.exe que vem junto na instalação.

  2. Execute o instalador

    3. Quando chegar na seção de escolha do local no qual o jogo será instalado, escolha um local livre, como por exemplo: A Pasta de Documentos ou a Pasta de Downloads.

Jogo ameaça executar e mais nada

Se quando você executa o jogo, parece que tudo vai correr bem, mas nada acontece, este é o seu problema. Para resolver isso, siga os passos abaixo:

  1. Reinicie o computador

    2. É necessário que o computador seja reiniciado após a instalação, para que tudo tenha efeito.

  2. Verifique a pasta do jogo

    3. Isso pode parecer simples, mas se um único arquivo estiver faltando, uma exceção é levantada, impedindo a boa execução do jogo.

    Caso tenha se certificado que há mudança nos arquivos, uma reinstalação deve resolver.

O jogo está travando

Se quando você executa o jogo, ele aparenta estar um pouco lento, a música de fundo fica oscilando, então este é o seu problema. Para resolver isso, siga os passos abaixo:

  1. Reabra o jogo

    2. Se é a primeira que você executa o jogo, é normal ele apresentar certa lentidão. Para resolver isso, simplesmente feche o jogo e abrá-o novamente.

  2. Feche programas pesados

    3. Programas potencialmente pesados podem sim, afetar o desempenho do jogo. Poderíamos citar, por exemplo: Google Chrome, Microsoft Edge, Editores de Vídeo, Steam, Spotify, YouTube Music entre outros.

    Vai depender das configurações da sua máquina, mas normalmente manter esses programas abertos durante a jogatina não é recomendado.

  3. Reinicie o computador

    4. Essa dica vai para quem costuma manter o computador na mesma sessão por dias. Não desligar o computador com certa frequência é uma prática não recomendada, já que isso retarda a velocidade de resposta do computador.

    Reveja novamente há quanto você reiniciou sua máquina. Se isso foi há mais de 3 dias, você deve reiniciá-lo.

Ainda tenho problemas

Se mesmo depois das dicas acima, seu ainda jogo não executa corretamente, por favor, siga os passos abaixo para que você possa identificar e reportar o erro:

  1. Execute o Prompt de Comando ou PowerShell

    2. Com o cursor sobre a bandeira do Windows (A qual você clica para acessar o menu iniciar), clique com o botão direito e escolha "Prompt de Comando" ou "PowerShell".

  2. Executar comandos

    3. Depois de aberto, digite os seguintes comandos que ajudarão a identificar o erro:

    1. cd "{local-onde-o-jogo-esta}"

      2. Você digitará este primeiro comando para adentrar na pasta onde o jogo foi instalado. Para que tudo ocorra bem, substitua o {local-onde-o-jogo-esta} pelo local onde você instalou o jogo ou onde ele está atualmente.

      Por exemplo: cd "C:\Users\myname\Documents\Jogos"

    2. MyAdventurousDino.exe

      3. Agora, já dentro da pasta na qual o jogo foi instalado, digite o comando acima. Isso vai executar o jogo e mostrar caso ocorra algum erro.
